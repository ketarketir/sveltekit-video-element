# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-22

### Added

- Initial release of sveltekit-video-element
- JW Player Video component with full configuration support
- Cloudflare Stream Video component with signed URL support
- Video.js component with plugin system
- HLS Video component with hls.js integration
- DASH Video component with dash.js integration
- Mux Video component with Mux Data analytics support
- Cloudinary Video component with transformation API
- Custom Video component for custom player integration
- Full TypeScript support with comprehensive type definitions
- SSR (Server-Side Rendering) support
- All standard HTML5 video events forwarding
- Utility functions for URL building and script loading
- Comprehensive documentation with examples
- MIT License

### Features

- Tree-shakeable exports
- Responsive video support
- Quality level switching for HLS/DASH
- Automatic error recovery for streaming protocols
- Cross-origin resource sharing (CORS) support
- Playsinline support for mobile devices
- Poster image support
- Multiple video format support
- Player instance export for advanced control
- Event forwarding to parent components
- Browser environment detection
- Dynamic script and stylesheet loading

### Documentation

- Comprehensive README with usage examples
- EXAMPLES.md with advanced use cases
- TypeScript type definitions
- Inline code documentation
